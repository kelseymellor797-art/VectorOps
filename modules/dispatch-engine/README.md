# @vectorops/dispatch-engine

Pure domain logic for dispatching tow calls to units.

## Overview

This module contains framework-agnostic services and utilities for dispatch operations. It has no HTTP, database, or external service dependencies — only `@vectorops/core`.

## Structure

- **services/** — Interfaces for dispatch operations
- **utils/** — Helper functions for distance calculation and unit availability

## Dependencies

- `@vectorops/core`
